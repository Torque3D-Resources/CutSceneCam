Custom Install Instructions:

A custom install should be used when integrating the Cut Scene Cam into a pre-existing project. The instructions in this section are mandatory for a custom install, optional instructions are below. These instructions are for a new project created using the "Full" template in T3D 3.10.0.

After you have fully finished the installation process, open up a new mission within that project and press "alt s" to open the Cut Scene Cam Editor. Read the text and tooltips within the editor and you should have your camera up and running in no time. If you need further assistance, check the Usage Instructions.


Step 1:

Backup all files that are going to be changed. They are:

source/gui/game/guiFadeinBitmapCtrl.h
source/gui/game/guiFadeinBitmapCtrl.cpp
source/T3D/pathCamera.h
source/T3D/pathCamera.cpp
scripts/server/scriptExec.cs



Step 2:

In the source folder, write over the stock files with the files that came with the CSC:

source/gui/game/guiFadeinBitmapCtrl.h
source/gui/game/guiFadeinBitmapCtrl.cpp
source/T3D/pathCamera.h
source/T3D/pathCamera.cpp



Step 3:

Recompile the engine. If you are unsure how to do this, check here for further instructions.



Step 4:

File- scripts/server/scriptExec.cs
After the line:

// Load turret support scripts
exec("./turret.cs");

Add:

exec("./CSC_userValues.cs");


Step 5:

In the folder: scripts/server/
Add the following files:

CSC_pathCamera.cs
CSC_pathManagement.cs
CSC_userValues.cs



Step 6:

Add the following editor files to the correct locations:

tools/worldEditor/gui/CSC_Editor.gui
tools/worldEditor/scripts/CSC_Editor.cs
tools/worldEditor/scripts/CSC_PathEditor.cs



Optional Custom Install Instructions:

These instructions are an optional part of a custom install. This will add the Cut Scene Cam Editor to the bottom of your "Edit" menu within the mission editor. If you're having problems while doing a custom installation, try skipping this part. This step is not included in the quick install files.


Step 1:

File- tools/worldEditor/scripts/menus.ed.cs
Function- EditorGui::buildMenus()
After the line:

      item[16] = "PostEffect Manager" TAB "" TAB "Canvas.pushDialog(PostFXManager);";

Add:

      item[17] = "-";
      item[18] = "Cut Scene Cam Editor" TAB "Alt S" TAB "toggleCSCEditor(1);";
